# Asset Attribution

This prototype uses procedural placeholder art generated at runtime for all gameplay objects.
No third-party binary art or audio files are bundled yet, so the game runs without missing-asset errors.

Planned free packs for visual/audio replacement:
- Kenney Platformer Pack (CC0): https://kenney.nl/assets/platformer-pack-redux
- Kenney UI Pack (CC0): https://kenney.nl/assets/ui-pack
- ZapSplat free SFX (license terms apply): https://www.zapsplat.com

When importing these packs, keep final license text and source links in this file.
